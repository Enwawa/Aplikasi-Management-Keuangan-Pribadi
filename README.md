# Aplikasi-Management-Keuangan-Pribadi
MANAGEMENT KEUANGAN PRIBADI
Aplikasi Manajemen Keuangan Mahasiswa merupakan aplikasi berbasis Java Object-Oriented Programming (PBO) yang dirancang untuk membantu mahasiswa dalam mengelola keuangan pribadi secara mandiri. Aplikasi ini digunakan untuk mencatat pemasukan dan pengeluaran, menghitung saldo secara otomatis, serta menyajikan laporan keuangan sederhana.
